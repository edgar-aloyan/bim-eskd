# 001_server_container

## Описание

Контейнерный дата-центр для майнинга на базе 40-футового ISO-контейнера (ISO 668, тип 1AA). 240 ASIC-майнеров Antminer S21 XP на 6 ярусах стальных полок с воздушным охлаждением (адиабатические пэды + вытяжные вентиляторы).

## Раздел

ЭОМ (электрооборудование и электроосвещение)

## Основные параметры

| Параметр | Значение |
|---|---|
| Контейнер | 12.192 × 2.438 × 2.591 м, стенки 43/80 мм |
| Вентзазор | 600 мм (контейнер на z=+0.6) |
| Фундамент | 8 буронабивных ЖБ столбиков ∅400, h=1600 мм |
| Каркас | 38 стоек + 4 ригеля, модуль 600×600 мм |
| Полки | 6 ярусов, лист 10920×498×3 мм, шаг 398 мм |
| Майнеры | 240 × Antminer S21 XP (449×219×293 мм, 3645 Вт) |
| Мощность | 875 кВт (≈0.884 МВА, PF>0.99) |
| Трансформатор | ТРС-160 (0.8/0.4 кВ, 160 кВА) в пролёте щитовой |
| Напряжение | 800 В линейное (класс НН до 1 кВ) |
| Изоляция L↔PE | Изоляторы SM-51 (creepage 51 мм, PD3) |
| Охлаждение | Адиабатическое: фильтр → пэд → демистер → холодный коридор |

## Состав модели (320 продуктов)

- 4 стены (IfcWall) — контейнер
- 2 плиты (IfcSlab) — пол + крыша
- 1 дверь (IfcDoor) — двустворчатая 2.34×2.28 м
- 8 фундаментов (IfcFooting) — буронабивные сваи
- 54 элемента каркаса (IfcMember) — стойки, ригели, рамки полок
- 6 полок (IfcPlate) — стальные листы
- 240 майнеров (IfcBuildingElementProxy) — Antminer S21 XP
- 1 трансформатор (IfcBuildingElementProxy) — ТРС-160
